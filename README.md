# 📝 React Task Manager

A modern task management web application built with React. This project was developed as part of a hands-on learning journey to understand React fundamentals, component-based architecture, state management, routing, and browser storage.

---

## 🚀 Features

### Task Management
- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🔄 Undo completed tasks
- ❌ Delete tasks

### Data Persistence
- 💾 Automatically saves tasks using Local Storage
- 🔄 Tasks remain available after page refresh
- 🌐 Data persists between browser sessions

### Navigation
- 🏠 Home Page
- 📋 Tasks Page
- ℹ️ About Page

## ⚙️ How It Works

### Adding Tasks
Users can enter a task and add it to the task list.

### Editing Tasks
Existing tasks can be modified without deleting and recreating them.

### Completing Tasks
Tasks can be marked as completed and toggled back to pending status.

### Deleting Tasks
Tasks can be permanently removed from the list.

### Persistent Storage
The application automatically saves task data in Local Storage so tasks remain available even after refreshing the page.
